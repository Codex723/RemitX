# Contributing to RemitX

We welcome contributions! RemitX is part of the **Drips Wave** ecosystem, focusing on programmable money and efficient routing.

## Development Workflow

1. Fork the repo and create a branch.
2. Ensure your code follows the TypeScript best practices.
3. Submit a PR with a clear description of changes.

## Issue Levels

- **Good First Issue**: Small bug fixes or type definitions.
- **Medium**: Logic improvements for the Path Router.
- **Hard**: Real-time monitoring and advanced Soroban contract integration.

Thank you for building the future of remittances!
